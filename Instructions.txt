 Ball Catcher Game - Setup Instructions

1. CREATE PROJECT FOLDER
   - Create a new folder named "my-html5-game"

2. CREATE FILES
   - Create three files in the folder:
     - index.html
     - style.css
     - game.js

3. COPY CODE
   - Copy the HTML code into index.html
   - Copy the CSS code into style.css
   - Copy the JavaScript code into game.js

4. TEST LOCALLY
   - Open index.html in your web browser to test the game
   - Make sure all features work properly

5. SET UP GIT (if not already done)
   - Open terminal/command prompt
   - Navigate to your project folder:
     cd path/to/my-html5-game

6. INITIALIZE GIT REPOSITORY
   - Run these commands:
     git init
     git add .
     git commit -m "Initial commit: Add Ball Catcher game"

7. CREATE GITHUB REPOSITORY
   - Go to GitHub.com and log in
   - Click the "+" icon → "New repository"
   - Name it "my-html5-game"
   - Make it public
   - Do NOT initialize with README (you already have files)

8. CONNECT LOCAL REPOSITORY TO GITHUB
   - Copy the commands GitHub shows you (they will look like):
     git remote add origin https://github.com/naveenr810953/my-html5-game.git
     git branch -M main
     git push -u origin main

9. ENABLE GITHUB PAGES
   - Go to your repository on GitHub
   - Click "Settings" → "Pages"
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click "Save"

10. ACCESS YOUR LIVE GAME
    - Your game will be available at:
      https://naveenr810953.github.io/my-html5-game/
    - It may take a few minutes to deploy

TROUBLESHOOTING:
- If game doesn't load: Check browser console for errors (F12)
- If assets don't load: Ensure file paths are correct
- If GitHub Pages doesn't work: Make sure main HTML file is named "index.html"

NEXT STEPS:
- Customize the game by modifying game.js
- Update the README.md with your information
- Share your game link with others!
